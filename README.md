Bem-vindo ao projeto **Calculadora Web**! Esta é uma ferramenta poderosa e intuitiva projetada para realizar operações matemáticas rápidas diretamente no seu navegador. Com uma interface limpa e foco em usabilidade, ela combina a robustez do JavaScript Vanilla com a agilidade do framework Bootstrap.

<h2 id="funcionalidades">4. 🧩 Funcionalidades Principais</h2>

Nossa ferramenta oferece tudo o que você precisa para cálculos cotidianos:

| Funcionalidade | Descrição |
| :--- | :--- |
| ➕ **Operações Aritméticas** | Soma, subtração, multiplicação e divisão precisas. |

<h2 id="destaques-tecnicos">5. 💻 Destaques Técnicos</h2>

A lógica por trás dos cálculos envolve conceitos importantes de computação:

### 📐 Lógica de Operadores
Implementação de captura de eventos (`Event Listeners`) em cada botão, permitindo que o JavaScript interprete strings numéricas e as converta em operações lógicas reais.



### 🔄 Gestão de Visor
Desenvolvimento de uma função de atualização de tela que gerencia a concatenação de dígitos e impede a inserção de múltiplos pontos decimais inválidos.

<h2 id="faq">8. 🧠 Perguntas Frequentes</h2>

<details>
<summary><strong>A calculadora respeita a ordem de precedência ❓</strong></summary>
<p>🧮 <strong>Resposta:</strong> Sim! A lógica foi construída para processar as expressões matemáticas respeitando as regras aritméticas padrão.</p>
</details>

<details>
<summary><strong>Posso usar o teclado numérico do computador ❓</strong></summary>
<p>⌨️ <strong>Resposta:</strong> Atualmente a interação é via clique, mas o suporte ao teclado (Keydown events) é uma funcionalidade planejada para a versão 2.0.</p>
</details>

<details>
<summary><strong>Como evitar a divisão por zero ❓</strong></summary>
<p>🛡️ <strong>Resposta:</strong> O sistema possui uma validação básica que impede erros fatais no script ao tentar realizar operações matematicamente impossíveis.</p>
</details>
